# Sedona

## Run Locally

Clone the project

```bash
  git clone https://github.com/shock-dev/barbershop.git
```

Go to the project directory

```bash
  cd barbershop
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```
